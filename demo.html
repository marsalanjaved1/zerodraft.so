Product Requirements Document (PRD)

Status: Draft Priority: P0

This document outlines the technical and functional requirements for the new Engineering Workspace module. The goal is to unify documentation, code exploration, and AI assistance into a single view.

# Core Features

  • Three-Column Layout: A responsive grid system that handles file navigation, main editing canvas, and auxiliary tools simultaneously.
  • Context-Aware AI: The right sidebar must have read-access to the open document to provide relevant suggestions.
  • Real-time Collaboration: WebSocket based cursor tracking and diff syncing using OT (Operational Transformation).

# API Contract

The backend endpoints needed for the file explorer hydration.
